Эмм, ну это типо изначально проект, где тестировались мои 3 новые технологии: 
- TypeScript
- PrismaORM
- EJS 

И заспойлерю, я использовал их, причем в большом количестве! Почти все написано на тайпскрипт, 
все запросы к базе данных идут через PrismaORM. Ejs использовался для постов, меню, профиля
и профиля других людей.

В папке src(dist) есть файлы 
- check (Это проверка всяческой информации, например логина)
- controllers (Там находится основная логика приложений)
- middleware (там мидлвары)
- path (там находятся как бы ссылки в директории на views(папку с html и ejs))
- routes (там находятся роуты, которые перенаправляют в контроллеры)
- service (там находится логика для базы данных. Чаще используется админом)
- server.ts(js) (там находится основной сервер на express)

Еще отдельно есть директория views, там находятся файлы html и ejs

Теперь про каждую отдельно.

- check - Там хранятся 5 файлов, в каждом функция для проверки возраста,
гендера, информации, логина и пароля.

Логин используется для проверки еще и username, тоесть и для псевдонима.
На моем сайта логин можно выбрать при регистрации, а псевдоним менять.
info так-же используется не только для описания профиля, но и для постов.
